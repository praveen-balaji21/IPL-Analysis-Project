 IPL Data Analysis Project
Project Overview:

This project performs Exploratory Data Analysis (EDA) on IPL match datasets using Python.
The analysis focuses on team performances, toss impact, scoring trends, venue analysis, and match-winning patterns.

Technologies Used:

Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook.

Dataset Used:

The dataset contains IPL match information including:

Teams
Match winners
Toss winners
Venues
First innings scores
Second innings scores
Powerplay scores.


Player of the Match awards:

Data Cleaning

Exploratory Data Analysis:

Top winning teams

Key Insights:

Rajasthan had the highest number of wins in the dataset.
Teams winning the toss had a slight advantage in matches.
Average first innings scores were higher compared to second innings scores.
Certain venues favored batting-first teams.
Powerplay performance strongly influenced match outcomes.
